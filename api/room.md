## Room api

#### Create a room

```json
{
    "cmd": "create_room",
    "token": "efd0c3b8-35bb-4cd1-85db-9248d45d8afa",
    "name": "to user name",
    "description": "room description",
    "subject": "room subject"
}
```

#### Delete a room

```json
{
    "cmd": "del_room",
    "token": "efd0c3b8-35bb-4cd1-85db-9248d45d8afa",
    "room_id": "room id"
}
```
